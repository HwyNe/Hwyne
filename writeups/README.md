# CTF ve Bug Bounty Writeup'ları

Bu dizin, çözülen CTF challenge'larının ve keşfedilen güvenlik açıklarının detaylı raporlarını içerir.

## Dizini Düzenleme Kuralı
- `/ctf/`: Yarışma bazlı (örn. HackTheBox, TryHackMe, PicoCTF) klasörleme.
- `/bug-bounty/`: Platform veya zaafiyet bazlı raporlar.

## Örnek Rapor Formatı
1. **Zafiyet Adı**
2. **Etkilenen Sistem**
3. **Şiddet (Severity)**
4. **Keşif Süreci**
5. **Sömürü (Exploitation)**
6. **Çözüm Önerisi**
